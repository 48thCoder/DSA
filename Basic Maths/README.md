# Basic Maths

This folder contains beginner-friendly **Basic Maths** problems and solutions for Data Structures and Algorithms (DSA) interview preparation.

## 📌 Topics Covered

Typical topics in this section include:

- Counting digits
- Reversing numbers
- Palindrome check (number)
- GCD / HCF and LCM
- Armstrong numbers
- Prime number check
- Divisor counting and printing divisors
- Euclidean algorithm basics

## 🎯 Learning Goals

By practicing these problems, you should be able to:

- Build comfort with number manipulation
- Improve mathematical logic used in coding interviews
- Write efficient solutions with optimized time complexity
- Understand edge cases involving integers and limits

## 🛠️ Language

All solutions in this repository are written in:

- **C++**

## 📂 Folder Structure

Problems are organized topic-wise.  
This directory focuses only on **Basic Maths** fundamentals.

## ✅ How to Use

1. Pick a problem.
2. Try solving it on your own first.
3. Compare your approach with the provided solution.
4. Re-implement the logic without looking.
5. Practice time and space complexity analysis.

## 🚀 Recommended Practice Flow

- Start with easy number-based problems.
- Move to divisor and prime-related questions.
- Practice GCD/LCM and Euclidean algorithm thoroughly.
- Revise by solving previously attempted questions without hints.
