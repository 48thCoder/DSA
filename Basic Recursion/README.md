# Basic Recursion

A beginner-friendly collection of classic recursion problems in C++ to build strong problem-solving foundations for Data Structures & Algorithms interviews.

## 📌 What You’ll Learn

This folder focuses on the **core recursion patterns**:

- Defining clear base cases
- Breaking problems into smaller subproblems
- Understanding recursive call stacks
- Building intuition for backtracking-style thinking
- Comparing recursive and iterative approaches

## 🧠 Common Problem Types in Basic Recursion

Typical problems in this section may include:

- Print numbers from `1 to N` / `N to 1`
- Sum of first `N` natural numbers
- Factorial of a number
- Power/exponentiation (`a^b`)
- Fibonacci number generation
- Reverse a string / array recursively
- Check palindrome using recursion
- Recursive binary search (intro level)

## 🛠️ Prerequisites

Before starting, you should know:

- C++ functions and parameters
- Conditional statements (`if`, `else`)
- Basic loops
- Arrays / strings fundamentals

## ▶️ How to Run C++ Files

From the repository root (or this folder), compile and run:

```bash
g++ filename.cpp -o output
./output
```

Example:

```bash
g++ factorial.cpp -o factorial
./factorial
```

## ✅ Suggested Learning Order

1. Start with simple print-based recursion
2. Move to mathematical recursion (`factorial`, `sum`, `power`)
3. Practice recursive checks (`palindrome`, `sorted array check`)
4. Try parameterized vs functional recursion styles
5. Revisit problems and dry-run by hand

## 🧪 Practice Tips

- Always write the **base case first**
- Dry run with small inputs (`n = 2, 3, 4`)
- Track each function call and return value
- Watch out for:
  - missing base case
  - incorrect recursive step
  - stack overflow for large inputs

## 📚 Why This Matters

Recursion is a foundation for advanced topics such as:

- Backtracking
- Divide and conquer
- Trees
- Dynamic programming

Mastering these basics makes advanced DSA significantly easier.
