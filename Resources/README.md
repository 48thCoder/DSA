
# Resources

This folder contains notes, reference materials, cheat sheets, and learning resources that support my Data Structures and Algorithms (DSA) journey.

## Contents

* **Notes** – Topic-wise notes and explanations.
* **Cheat Sheets** – Quick revision guides for important concepts.
* **STL Reference** – Commonly used C++ STL containers, algorithms, and functions.
* **Complexity Analysis** – Time and space complexity references.
* **Algorithms** – Important algorithms and their implementations.
* **Problem-Solving Patterns** – Common approaches such as Sliding Window, Two Pointers, Binary Search, Recursion, Dynamic Programming, etc.

The goal of this folder is to maintain a centralized collection of resources for quick revision, concept clarification, and interview preparation.
